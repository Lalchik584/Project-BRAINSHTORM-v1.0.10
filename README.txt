Quiz Master Platform 

Профессиональная платформа для проведения интерактивных квизов в реальном времени с поддержкой QR-кодов, live-результатов и автоматическим ведением статистики.

Возможности

Для учителей:
- Создание квизов с любым количеством вопросов
- Поиск по базе созданных квизов
- Управление сессиями в реальном времени
- Live-статистика ответов студентов
- Таблица лидеров с автоматическим обновлением
- QR-коды для быстрого подключения студентов
- Контроль времени на каждый вопрос

Для студентов:
- Простое подключение по 6-значному коду
- Поддержка QR-кодов для мгновенного входа
- Таймер обратного отсчета для каждого вопроса
- Прогресс-бар прохождения квиза
- Детальные результаты после завершения
- Автоматический переход между вопросами

Быстрый старт
Требования:
- Node.js 18+
- npm или yarn

Установка:

1. Клонируйте репозиторий:
git clone https://github.com/ваш-username/quiz-master.git
cd quiz-master

2.Установите зависимости:
npm install

3. Запустите сервер:
node server.js

4. Откройте в браузере:
http://localhost:5000


Структура проекта
quiz-master/
├── server.js              # Основной серверный файл

Использование

1. Создание квиза
1. Перейдите на панель учителя: http://localhost:5000/teacher.html
2. Введите название квиза
3. Выберите количество вопросов (1-20)
4. Заполните вопросы и варианты ответов
5. Нажмите "Create Quiz"
6. **Сохраните артикул квиза** для последующего использования

2. Запуск сессии
1. Найдите созданный квиз через поиск
2. Нажмите "Start Quiz Session"
3. **Запомните 6-значный код сессии**
4. Поделитесь QR-кодом или ссылкой со студентами
5. Нажмите "Start Quiz" когда все присоединятся

3. Подключение студентов
1. Студенты открывают: http://localhost:5000/student.html
2. Вводят 6-значный код сессии
3. Вводят свое имя
4. Нажимают "Join Session"
5. Ждут начала квиза

Консольные команды
После запуска сервера доступны команды:
startquiz XXXXXX     # Запустить квиз в сессии XXXXXX
listsessions        # Показать активные сессии
endquiz XXXXXX      # Завершить квиз
nextquestion XXXXXX # Перейти к следующему вопросу

Технические особенности
Серверная часть:
- Express.js для обработки HTTP-запросов
- Socket.IO для real-time коммуникации
- UUID для генерации уникальных идентификаторов
- QR-image для генерации QR-кодов
- Файловая БД для хранения квизов

Клиентская часть:
- Vanilla JavaScript без зависимостей
- Responsive дизайн с CSS Grid/Flexbox
- WebSocket для live-обновлений
- Прогрессивная загрузка вопросов
